Scrollspy

Scrollspy in Bootstrap is a JavaScript plugin that automatically highlights the active navigation link based on the current scroll position of the page. It's a useful tool for improving usability and user experience.

Scroll the area below the navbar and watch the active class change. Open the dropdown menu and watch the dropdown items be highlighted as well.
